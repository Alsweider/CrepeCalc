# CrepeCalc
Berechnet die Zutatenliste für Crêpes anhand der vorhandenen Menge.

# Anleitung

Die Menge der gewünschten Zutat ändern und mit der Eingabetaste bestätigen. Die anderen Zutatenmengen werden automatisch berechnet.
